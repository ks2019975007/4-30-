나의 컴퓨터(가상 컴퓨터 견적 뽑기)



1.목적(어떤 용도의 컴퓨터)

전공 공부를 위한 파이썬이나 리눅스 등의 프로그램을 사용할 수 있고, 오버워치와 롤 정도의 게임을 돌릴 수 있는 사양의 게임용 컴퓨터가 목적이다.

2. 컴퓨터 각 부품 기능, 가격, 선정 이유

2-1. cpu
기능 : 컴퓨터의 정중앙에서 모든 데이터를 처리하는 장치라는 뜻이다. CPU는 컴퓨터의 두뇌에 해당하는 것으로서, 사용자로부터 입력 받은 명령어를 해석, 연산한 후 그 결과를 출력하는 역할을 한다.
제품명 : 인텔 코어i5-8세대 8500 (커피레이크) (정품)
가격 : 262,480원
선정 이유 : 코어나 쓰레드가 많아 게임 프레임이나 다중 작업 환경에서 좋고, 옵션타협만 하면 대부분의 게임들을 즐기는데 무리가 없는 사양이다. 쿨러가 장착되어 있어서 따로 사지 않아도 되고, 저전력 시스템으로 아주 경제적이며 6개의 코어로 더욱 빠른 작업이 가능하다.

2-2. 메인보드
기능 : CPU나 그래픽카드, 하드디스크 등이 PC의 성능을 좌우하는 요소라 한다면 메인보드는 이들 구성품 간의 호환성이나 차후 기능 확장의 범위를 정하며, 궁극적으로는 PC 전반의 안정적인 동작 여부를 좌우하는 요소라 할 수 있다.
제품명 : ECS DURATHON2 B360 TIGER-T9 코잇
가격 : 72,850원
선정 이유 : 중가 게이밍 pc에 어울리는 보드라고 생각했고 디자인이 깔끔해서 선정하였다.

2-3. 메모리
기능 : 주기억장치
제품명 : 삼성전자 DDR4 8G PC4-21300 (정품)
가격 : 47,170원
선정 이유 : 이 제품은 이전 19200 모델과 비교해서 동작 클럭의 성능이 올라서 선정하게 되었다.

2-4. 그래픽카드
기능 : 컴퓨터에서 처리되는 과정 및 결과를 눈으로 보기 위해서는 모니터가 필요한데, 모니터는 컴퓨터 내부의 그래픽 카드를 통하여 화면에 정보를 싣게 된다. 즉, 그래픽 카드에서 만들어진 신호가 모니터에서 빛으로 변환되어 인간의 눈으로 이해되는 것이다.
제품명 : 갤럭시 GALAX 지포스 GTX1060 개간지 EXOC D5 3GB
가격 : 266,000원
선정 이유 : 6GB는 30만원 초중반대인것에 비해, 3GB 버전은 성능이 10% 이내로 차이가 나는데도 20만원 중후반대로 상대적으로 가성비가 좋다. 그래서 Full HD이상 고해상도를 쓰는 것이 아닌 온라인 게임 위주로 사용하는 나에게 가성비가 좋은 메인스트림 그래픽카드 같다.

2-5. SSD
기능 : 반도체를 이용하여 정보를 저장하는 장치이다. 하드디스크드라이브에 비하여 속도가 빠르고 기계적 지연이나 실패율, 발열·소음도 적으며, 소형화·경량화할 수 있는 장점이 있다.
제품명 : ADATA Ultimate SU800 SHT (256GB)
가격 : 51,000원
선정 이유 : 3년의 A/S를 지원하여 신뢰할 수 있으며, 스펙적인 부분에선 기존 2차원 방식 MLC 방식에 비해 약 50% 정도의 전력소모를 줄였으며, TLC의 충분한 안정성과 성능은 오랜 시간 동안 입증을 받아왔기에 5만원대의 가격으로 충분한 가성비를 갖춘 제품이라고 생각한다.

2-6. 케이스
기능 : 컴퓨터 케이스(영어: computer case)는 컴퓨터의 주요 부품들을 보호한다.
제품명 : ABKO SUITMASTER 331S 스피릿 강화유리 LAYER
가격 : 77,000원
선정 이유 : 통풍이 잘 되고 쿨링 쪽에 신경을 많이 썼다고 느꼈고 led기능을 온오프 할 수 있어서 선정하게 되었다.

2-7. 파워
기능 : 전원을 on/off 한다.
제품명 : 쿨러마스터 MWE 500W 80PLUS Standard 230V EU
가격 : 42,500원
선정 이유 : 파워 선택에는 저소음에 중점을 두었기 때문에 고효율 저소음 HDB 베어링 120mm 쿨링팬이 장착되었고, 또한 A/S 기간이 5년이라는 점도 굉장히 신뢰도가 높아 선정하게 되었다.

2-8. 모니터
기능 : 컴퓨터에서 처리한 결과를 보여주는 출력 장치이다.
제품명 : 뷰소닉 VA2419 아이케어유 무결점
가격 : 139,000원
선정이유 : 가격이 굉장히 저렴하고 디자인과 크기가 무난히 마음에 들었다. 또한 뷰소닉의 서비스와 보증된 품질이 가장 큰 선정 이유였다.

2-9. 키보드
기능 : 주로 이메일이나 워드 프로세서 등에서 숫자 및 문자를 입력할 때 많이 쓰이며, 게임이나 각종 애플리케이션에서도 유용한 단축키와 특수 명령 기능을 제공한다.
제품명 : 씽크웨이 CROAD P204 LED 게이밍 키보드 마우스 세트
가격 : 11,500원
선정이유 : 키 감이 좋고 한 개씩 누를 때 마다 불이 들어와서 이쁘다.

2-10. 마우스
기능 : 컴퓨터 입력장치의 일종이다.
제품명 : 씽크웨이 CROAD P204 LED 게이밍 키보드 마우스 세트
가격 : 11,500원
선정이유 : 게임이 주 목적이기 때문에 마우스에 사이드 버튼이 있어서 선정하게 되었다.

2-8. 총 견적 가격
262,480원+72,850원+47,170원+266,000원+51,000원+77,000원+42,500원+139,000원+11500원 = 969,500원

3.호환성 테스트

CPU 인텔 코어i5-8세대 8500 (커피레이크) (정품)과 메인 보드 ECS DURATHON2 B360 TIGER-T9 코잇의 소켓이 인텔(소켓1151v2)로 같기 때문에 호환이 가능하다. (또한 그래픽 카드는 4세대 이상 제품이라면 모두 호환이 가능하다.)

4.조립후기

컴퓨터에 대해 잘 몰랐는데 이번에 과제를 하면서 컴퓨터에 어떤 부품이 들어가는지와 기능에 대해 자세히 알게 되었다. 특히 마음에 든다고 해서 사도 호환이 안 되면 사용이 안 된다는 점이 가장 신기했다. 항상 견적이 짜져 있는대로 삼성이나, LG에 가서 대충 설명만 듣고 사기만 했지 이렇게 제대로 견적을 짜본적은 처음이라 어렵기도 했고 재밌기도 했다. 앞으로 컴퓨터를 살 일이 있을 때 이번에 한 견적이 크게 도움이 될 것 같다.
